# Titanic Survival Analysis Dashboard using Microsoft Excel

## 📌 Project Overview

This project presents an interactive **Titanic Survival Analysis Dashboard** developed using **Microsoft Excel**. The dashboard analyzes passenger survival patterns based on various demographic and travel-related factors, providing meaningful insights through interactive visualizations.

## 🎯 Objective

The objective of this project is to explore the Titanic dataset, identify the key factors affecting passenger survival, and present the findings in an easy-to-understand interactive dashboard.

## 🛠️ Tools & Features Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Interactive Dashboard
- Data Cleaning
- Data Analysis

## 📊 Dashboard Includes

- Total Passengers
- Total Survivors
- Total Deaths
- Overall Survival Rate
- Survival Rate by Gender
- Survival Rate by Passenger Class
- Survival Rate by Fare Group
- Survival Rate by Age Group
- Survival Rate by Boarding Port
- Survival Rate by Family Status (Alone vs With Family)

## 📈 Key Insights

- Female passengers had a significantly higher survival rate than male passengers.
- First-class passengers had the highest survival probability.
- Children had better survival chances than adults.
- Passengers who paid higher fares generally showed higher survival rates.
- Survival patterns varied based on family status and boarding location.
