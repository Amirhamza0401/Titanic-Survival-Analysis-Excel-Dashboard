Titanic Survival Analysis Dashboard (Excel)
Project Overview

This project analyzes the Titanic passenger dataset using Microsoft Excel. The objective is to explore survival patterns based on passenger demographics and create an interactive dashboard for business-style reporting.

Survival Rate by Fare Group
Survival Rate by Age Group
Survival Rate by Boarding Port
Survival Rate by Family Status
Interactive Slicers for filtering data
Key Insights
Female passengers had a significantly higher survival rate than males.
First-class passengers had the highest survival rate.
Children showed better survival chances than adults.
Higher fare passengers generally had better survival rates.
Passengers traveling alone had different survival patterns compared to those traveling with family.
